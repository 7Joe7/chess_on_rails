Class Player
In: app/models/player.rb
Parent: ActiveRecord::Base

A Player is a participant in any Chess match, or simply a known account of someone. It was used as the Restful Authentication user model so @current_player is used instead of @current_user. Other authentication types like facebook, or OpenID, when implemented will simply map their identities onto instances of Player in this application
